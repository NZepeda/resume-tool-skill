# Command: improve bullets

## Purpose

Rewrite selected bullets for clarity, impact, and ATS alignment.

## Required Inputs

- The bullets to improve
- Target role or JD (optional but recommended)

## Workflow

1. Identify the role context if provided.
2. Rewrite each bullet using the Bullet Rewrite Formula.
3. Flag missing metrics and evidence gaps.

## Output Blueprint

- Revised Bullets
- Notes (why changes were made)
- Missing Inputs (if blocked)

## State Updates

- None, unless requested to update the Master Resume section.

## Evidence Checks

- Do not invent metrics or scope; flag missing data.

## Example Input

Bullets:
- Improved onboarding flow for new users.
- Built dashboards for product analytics.

Target role: Product Manager

## Example Output

Revised Bullets:
- Redesigned onboarding flow to reduce time-to-first-action by 30% across 3 steps; validated with A/B tests.
- Built product analytics dashboards used weekly by 20+ stakeholders to track activation and retention.

Notes:
- Added outcome metrics; confirm actual values.

## Missing Inputs

- "Paste the bullets you want improved."
- "What role or JD should these target?"
