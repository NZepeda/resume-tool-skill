# Calibration Checklist

Use this checklist to evaluate whether the scoring rubric aligns with real outcomes.

## Setup

- Select 3 resumes (early, mid, senior).
- Select 3 job descriptions across different industries.
- Run `ingest jd`, `tailor resume`, and `score` for each pairing.

## Score Review

- Compare ATS scores against actual keyword coverage and formatting.
- Compare recruiter scores against a human review.
- Identify any consistent over- or under-scoring patterns.

## Adjustments

- If ATS scores are too high, increase penalties for keyword gaps and formatting risks.
- If recruiter scores are too low, revisit impact clarity and differentiation weights.

## Pass/Fail Criteria

- ATS score should correlate with parseability and keyword alignment.
- Recruiter score should roughly match a human evaluator’s ranking.
- Any repeated discrepancy should produce a rubric tweak.
